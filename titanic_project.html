def survival_by_groups(data, group):
    '''
    Function takes the titanic dataset, a column name from the dataset and 
    returns a pandas series with the survival rates (%) for cognate groups within the specified column.
    
    data: titanic dataset (pandas data frame)
    group: name of columns from the titanic dataset for which the survival rates should be calculated (list)
    '''
    
    total_in_group = data.groupby(group, as_index = True).count()
    group.append("Survived")
    survived_in_group = data.groupby(group,as_index = True).count().reset_index(level = "Survived")
    
    survived = []
    for i in range(len(survived_in_group)):
        if survived_in_group["Survived"][i] == "survived":
            survived.append(i)
    
    return ((survived_in_group.iloc[survived]["PassengerId"]/total_in_group["PassengerId"])*100).fillna(0)

Statistical analysis of the group-specific survival rates¶

Now let's conduct statistical tests and evaluate the group-specific survival rates in a statistical context, i.e do the observed differences provide enough evidence for a significant correlation with the investigated groups? Because we have nominal (categorical) variables and want to test for correlation between variables, we will conduct a $\chi^2$ (chi-square) test for independence .

\begin{equation*} \chi^2 = \sum{\frac{f_o - f_e}{f_e}} \quad\quad\quad \text{$f_o$ = observed frequency} \quad\quad\quad \text{$f_e$ = expected frequency} \end{equation*}

Hypothesis
H0 (Null Hypothesis): There is no correlation between survival rate and one of the other variables 1. start harbour, 2.ticket class, 3. Gender. Both variables are independent
H1(Alternative Hypothesis): There is a correlation between survival rate and one of the other variables. The variables are not independent.

Parameters for the chi-square test
Survival by start harbour
critical statistic value (chi-square critical) = 5.991(alpha level = 0.05, degrees of freedom = 2)

Survival by ticket class
critical statistic value = 5.991(alpha level = 0.05, degrees of freedom = 2)

Survival by gender
critical statistic value = 3.841(alpha level = 0.05, degrees of freedom = 1)

def chi_square_independence_test(data, group):
    '''
    Function takes the titanic dataset, a column name from the dataset and returns the result of a 
    chi-square test for independence for a specified variable (column name) and the "Survived" variable from
    the dataset.
    
    data: titanic dataset (data frame)
    group: name of the column from the titanic dataset to be used as specified variable (string)
    '''
    # creates a contingency table of the observed frequencies of variables "Survived" and "Specified Column"
    contingency_table_observed = pd.crosstab(index = data["Survived"], columns = data[group], margins = True)
    contingency_table_observed.index = np.append(contingency_table_observed.index[:-1], "coltotal")
    contingency_table_observed.columns = np.append(contingency_table_observed.columns[:-1], "rowtotal")
    
    rowtotal_not_survived = contingency_table_observed["rowtotal"].loc["not survived"]
    rowtotal_survived = contingency_table_observed["rowtotal"].loc["survived"]
    total = contingency_table_observed["rowtotal"].loc["coltotal"]
    
    # creates a contingency table of the expected frequencies of variables "Survived" and "Specified Column"
    # expected values are based on the marginal frequencies for the total row and column entries
    expected_not_survived = (contingency_table_observed.loc["coltotal"]*rowtotal_not_survived)/total
    expected_survived = (contingency_table_observed.loc["coltotal"]*rowtotal_survived)/total
    contingency_table_expected = pd.DataFrame()
    contingency_table_expected = contingency_table_expected.append([expected_not_survived,expected_survived,contingency_table_observed.loc["coltotal"]], ignore_index = True)
    contingency_table_expected.index = contingency_table_observed.index
    
    # calculates the chi-square statistic
    chi_square = sum( (((contingency_table_observed-contingency_table_expected)**2)/contingency_table_expected).loc["survived"] ) + \
    sum( (((contingency_table_observed-contingency_table_expected)**2)/contingency_table_expected).loc["not survived"] )
    
    # calculate Cramer's V as measure of association between the tested variables
    min_dimension = min(len(contingency_table_observed.columns[:-1])-1,len(contingency_table_observed.index[:-1])-1)    
    cramers_v = np.sqrt(chi_square/(total*min_dimension))
    
    print ("chi-square statistic for correlation between survival rate and {0}: {1}".format(group,round(chi_square,2))+\
           "\nCramer's V: {}".format( round(cramers_v,2) ))

def age_classification(bins, data):
    ''' 
    Function takes a sequence of numbers with a specified step size defining the age groups and returns
    a titanic dataset extended by an age group column that contains age group entries for each pasenger with known
    age. If a passenger age is missing, it will be classified as NaN age group.
        
        
    data:   titanic dataset (pandas data frame)
    
    bins:   sequence of numbers that define the range of age groups (list or tuple).
            for example: [0,5,10,15] would group passengers in classes of age 0-5, 5-10 and 10-15, respectively
    '''
    
    # define left (bmin) and right border (bmax) for each bin and append as tuple to bins_grouped
    bins_grouped = []
    for i in range(len(bins)):
        try:
            bins_grouped.append((bins[i], bins[i+1]))
        except IndexError:
            break
    
    # 1. loop over the dataset and extract age for each passanger
    # 2. if passanger age is within range of any of the defined bins, define the age group and append it to age_group
    # 3. if age is missing (NaN), append NaN
    age_groups = []    
    for i in range(len(data)):
            
        age_passenger = data["Age"].iloc[i]
        
        if np.isnan(age_passenger):
            age_groups.append(np.nan)
        
        for bins in bins_grouped:
            
            bmin = bins[0]
            bmax = bins[1]   
            index_bin = "age {0}-{1}".format(bmin,bmax)
        
            # if the passenger age groups within the last bin, include age. Otherwise take only age < bmax
            # So the first and last bin include the minimum and maximum value 
            if bins == bins_grouped[-1]:
                if (age_passenger >= bmin and age_passenger <= bmax):
                    age_groups.append(index_bin)
                    
            else:
                if (age_passenger >= bmin and age_passenger < bmax):
                    age_groups.append(index_bin)
                    
                    
    age_groups = pd.Series(age_groups, name = "Age group")
    return pd.concat([age_groups, data], axis = 1)

    '''

    A -> deck A (topmost deck with cabins)

        if "D" in entry:
            return "D deck"
        if "E" in entry:
            return "E deck"
        if "F" in entry:
            return "F deck"

def deck_types_by_ticket_class(data):
    ''' 
    This function will take the titanic dataset with the one letter code for the "Cabin" column, group it 
    by ticket class and deck type ("Pclass" and "Cabin) and calculate the passenger counts for each group. 
    It will return a data frame with ticket class, cabin types and counts as columns, and rows representing 
    passenger counts for each ticket class and cabin type.
    
    
    (e.g.: 1, A, 15 corresponding to 15 passengers with ticket class 1 on deck type A
        
    data: titanic_csv with cabin entries converted to one letter code (pandas data frame)
    '''
   
    
    grouped = titanic_csv[["Pclass", "Cabin", "PassengerId"]].groupby(["Pclass", "Cabin"], as_index=False).count()
    
    # the set of deck types (in one letter code) is determined for the titanic dataset, against which the individual 
    # deck type sets of each ticket class are compared. If a deck type is missing from a ticket class, a value of 0 
    # is set for this deck type and a new entry is appended to the grouped titanic_csv (e.g.: 1, A, 0 corresponding
    # to 0 passengers with ticket class 1 on deck type A). 
    
    class1=set()
    class2=set()
    class3=set()
    for i in range(len(grouped)):
        if grouped.loc[i]["Pclass"] == "class 1 ticket":
            class1.add(grouped.loc[i]["Cabin"])
        elif grouped.loc[i]["Pclass"] == "class 2 ticket":
            class2.add(grouped.loc[i]["Cabin"]) 
        else:
            class3.add(grouped.loc[i]["Cabin"])    

    dict_cabins = {"class1":class1, "class2":class2, "class3":class3}
    all_categories = class1 | class2 | class3

    for ticket_class in dict_cabins:
            for category in all_categories:
                if category not in dict_cabins[ticket_class]:
                    missing = pd.DataFrame([[ticket_class, category,0]], columns=grouped.columns)
                    grouped = grouped.append(missing)
                    
    def transform(entry):
        if entry == "class1":
            return "class 1 ticket"
        if entry == "class2":
            return "class 2 ticket"
        if entry == "class3":
            return "class 3 ticket"
        else:
            return entry
        
    grouped["Pclass"] = grouped["Pclass"].apply(transform)    
    return grouped




def transpose(data,index_col,transpose_col):
    '''
    takes a data frame of the titanic dataset that was grouped by two variables and contains numerical values 
    for each group. Each group and the corresponding numerical value must be represented as a one row entry. So when
    using the groupby() function for data frames, the argument as_index must be set to False. The "PassengerId" column
    from the titanic dataset must be used to represent the numerical values. 
    It returns a transposed data frame with the entries set of one of the two variables used for grouping as columns
    and the entries set of the other variable used for grouping set as row index. 
    
    As such, the returned data frame can be directly used to plot the number of passengers as a function of the 
    two columns used for grouping (via .plot() method for data frames).
    
    e.g., data frame with columns "Pclass" (ticket class), "Cabin" (cabin type), "PassengerId" (counts) will be 
    converted into data frame with columns "A", "B" and index "class1", "class2"
    
    Pclass    |    Cabin   |   PassengerId           ------->                    A   |   B
    ----------------------------------------                          ---------------------
    class 1   |     A      |       15                                 class 1    15  |   25
    class 1   |     B      |       25                                 class 2    0   |   10
    class 2   |     A      |       0
    class 2   |     B      |       10
    
    
    index_col: name of column from the titanic dataset to be used to set row indexes (string)
    transpose_col: name of column from the titanic dataset to be used for setting columns in the transposed df (string)
    '''
    
    data = data.sort_values(by = [index_col, transpose_col])
    index = oset(data[index_col].values)
    columns = defaultdict(list)
    len_column = None
    max_len = None
    
    # 1. loop over the data frame, take entry in transpose_col as key in defaultdict and append corresponding val from 
    #"PassengerId" column
    # 2. monitor length of each defaultdict list and keep track of max length for any column
    # 3. loop over columns defaultdict and compare the length of each list with the max length
    # 4. if any column has less entries than column with most entries (len(column_i) < len(column_max)) append 0
    
    # This will make sure that transposed data frame will contain all groups with missing group values set as 0
    
    for i in range(len(data)):
        columns[data.iloc[i][transpose_col]].append(data.iloc[i]["PassengerId"])
        len_column = len(columns[data.iloc[i][transpose_col]])
        if max_len < len_column:
            max_len = len(columns[data.iloc[i][transpose_col]])

    for key in columns:
        if len(columns[key]) != max_len:
            i = max_len-len(columns[key])
            for repeat in range(i): 
                columns[key].append(0)
        
        
    return pd.DataFrame(columns, index = index)

Summary¶

Overall, the data shows that a passenger's survival probability is highly correlated to gender, age and ticket category, with gender showing the strongest association. Female passengers were almost 4x as likely as men to survive. One possible explanation is the higher priority in saving women, which is supported by the finding that children (age 0-10) as well were almost twice as likely to survive. This is known as the "Women and children first" honorary code in situations of abandoning ships (https://en.wikipedia.org/wiki/Women_and_children_first). However, higher survival probabilities also correlate with more expensive ticket classes, which might be explained by cabin locations being on upper decks of the ship. In this context I hypothesize that rescue boots are easier and faster accessible for cabins located on the upper half of the ship.
